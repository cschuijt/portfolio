Development projects

Gem of True Sight is a Telegram bot that lets users sign in with Steam so that it can fetch their Dota 2 match data, statistics and profile data and display it in any chat on request. It is written in Rails and combines several APIs, like the OpenDota API, the Steam API and of course the Telegram bot API. It can cache and paginate API results, ask for clarification when a user's input is unclear and respond to inline queries to post into any chat, even when the bot itself is not there. It also has a fully featured test suite and CI/CD pipeline.

It is currently live at dotagem.net and on Telegram as @dotagem_bot. The source code is available on GitHub.

Services hosted

Nightscout is an open-source glucose monitoring system "in the cloud". While proprietary systems exist to upload data from flash glucose monitors to the internet, these didn't have the functionality or portability that I was looking for, so I took to hosting my own instead. In the process, I learned how to set up MongoDB, which runs on my own server.

Access to my Nightscout instance is password-controlled (as it probably should be), but feel free to contact me if you would like a demonstration.
